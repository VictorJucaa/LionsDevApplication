# LionsDevApplication
EN - 

This is a project created for an application for an Internship/Development position. The project consists of delivering a responsive web page with at least one button/link that leads to an external page.

I chose Holy Grail Layout because of its simplicity combined with its versatility. Since the company is in the retail sector, I opted to display the main actions related to the workflow in the menu bar (such as options for products sold, consortiums, etc.).

The page was built entirely using only HTML and CSS.
